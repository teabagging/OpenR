Also, the smallest number b formed by any column is divisible by the largest number a formed by any row, but by the symmetry just proven, a is also the largest number formed by any column, so a geq b. Since b is divisible by a, we must have equality. Then all columns contain the same number - and hence all rows also contain the same number - which is only possible if all 16 cells contain the same digit. Conversely, for each d=1, ldots, 9, filling in all 16 cells with the digit d clearly gives a table meeting the required condition, so we have exactly 9 such tables, one for each digit.
